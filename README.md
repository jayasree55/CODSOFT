# Jayasree's Beginner AI & Java Projects

Welcome to my GitHub portfolio! This repository contains three beginner-stage Java projects that explore AI, algorithms, and interactive programming. These projects were developed during my learning journey at **#CodeSoft**.

---
 1. TicTacToeAI

Description:
TicTacToeAI is a Java-based game where a user can play Tic Tac Toe against an AI opponent. The AI uses logical decision-making to provide competitive gameplay, helping to learn algorithms and basic AI concepts.

Features:
- Play against an AI in Tic Tac Toe
- Win/draw detection and game logic
- Interactive console-based gameplay
- Beginner-friendly AI logic

Tech Stack:
- Java
- Console application

Run:javac TicTacToeAI.java
    java TicTacToeAI
